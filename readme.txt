# Необходима локальная сеть/использование одного Wi-Fi соединения.
# Необходим Python версии 3.10 или выше, также установленные библиотеки pygame и chess.
Для игры необходимо запустить файл ChessPython.pyw на обоих компьютерах; 
на одном нужно выбрать пункт меню Host Game, на втором выбрать пункт меню Join Game.
Для выхода из игры необходимо нажать на крестик в правом верхнем углу
После завершения партии игрокам предлагается начать новую партию(и сменить цвет фигур), для этого 
необходимо нажать на кнопку Reset после завершения игры.
